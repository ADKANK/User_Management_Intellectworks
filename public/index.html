API Documentation

Welcome to the API. Below are the available routes and their usage.

User Management

  • POST /api/register

    Register a new user.

  • POST /api/login

    User login.

  • PUT /api/edit

    Edit user details.

  • DELETE /api/delete

    Delete the user account.

Notes Management (Requires Authentication)

  • POST /api/notes

    Create a new note for the authenticated user.

  • GET /api/notes

    Retrieve all notes for the authenticated user.
